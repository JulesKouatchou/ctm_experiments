     Be10   = 10.0*Be10/28.965
     Be7    = 7.0*Be7/28.965
     Pb210  = 210.0*Pb210/28.965

     nh_5   = 0.001*nh_5
     nh_50  = 0.001*nh_50
     
**Addition of new tracers**:

     Be7s   = Be7
     Be10s  = Be10
